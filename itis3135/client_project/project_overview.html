1) Project Overview

Application/Purpose: Build a professional website for Deliverr Trucking LLC to promote its logistics services, highlight past projects, and make it easy for potential clients to contact the business for quotes

Intended Users: Logistics partners, freight brokers, and potential clients looking for dependable trucking services in the Southeast.

Overview of Content: The website will include an About page, Services page, Work Showcase (portfolio), and Contact form with dynamic interactivity using JavaScript and jQuery.

7) Dynamic Functionality

  • Get an Estimate (Contact) - HTML5 + JS validation and AJAX email submission via Formspree. Purpose: collect quote requests without a backend.
    Example: https://formspree.io/
  • FAQ Accordion (Services) - jQuery UI accordion for show/hide answers. Purpose: reduce scrolling and surface key info quickly.
    Demo: https://jqueryui.com/accordion/
  • Gallery Lightbox (Work Showcase) - image lightbox/modal. Purpose: view job photos without leaving the page.
    Demo: https://photoswipe.com/
  • Home Metrics (JSON Fetch) - Fetch a local metrics.json and render stats (e.g., on-time %). Purpose: demonstrate read-only dynamic content.
    Docs: MDN Fetch
  • Map Embed (Contact) - Google Maps Embed iframe. Purpose: help visitors find the business quickly.
    Docs: Google Maps Embed
