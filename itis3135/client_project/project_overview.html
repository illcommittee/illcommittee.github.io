Application/Purpose: Build a professional website for Deliverr Trucking LLC to promote its logistics services, highlight past projects, and make it easy for potential clients to contact the business for quotes

Intended Users: Logistics partners, freight brokers, and potential clients looking for dependable trucking services in the Southeast.

Overview of Content: The website will include an About page, Services page, Work Showcase (portfolio), and Contact form with dynamic interactivity using JavaScript and jQuery.

7) Dynamic Functionality

  • Gallery Lightbox (Work Showcase) - JavaScript-based photo expansion with captions.
  • FAQ Accordion (Services) - jQuery UI accordion for quick answers.
  • Contact Form Validation (Contact) - HTML5 + JavaScript form checks and AJAX submission.
  • JSON Fetch (Home) - Load a local metrics.json file to display company statistics.
